# 信息源优先级

本文件用于判断产业链选题、公司案例、市场热点和研报观点的证据质量。原则是：一手来源优先，二手来源辅助，低信任来源不能作为事实依据。

## S 级：一手来源

优先级最高。可作为事实核验核心依据。

- 公司公告
- 年报
- 半年报
- 招股书
- 交易所问询函和回复
- 公司官网
- 公司投资者关系材料
- 官方发布会
- 监管文件
- 政府或主管部门公开文件

使用方式：

- 用于确认业务范围、收入占比、客户表述、订单、产能、风险提示。
- 引用时尽量标注文件名称、日期和关键页码或段落。
- 不只截标题，要看正文语义和上下文。

## A 级：高质量二手来源

可作为辅助证据，但最好与 S 级来源交叉验证。

- 权威财经媒体
- 行业媒体
- 产业会议资料
- 行业协会报告
- 主流科技媒体
- 权威数据机构公开摘要

使用方式：

- 用于理解行业背景、技术路线、供需变化和市场规模。
- 不作为上市公司具体收入、订单、客户事实的唯一依据。

## B 级：辅助来源

只能作为线索，不能单独定论。

- 券商研报
- 机构会议纪要
- 产业专家访谈
- 数据库摘要
- 调研纪要

使用方式：

- 用于发现问题和关键词。
- 必须回到公告、年报、官网或其他公开来源验证。
- 不搬评级、目标价、受益标的名单和图表。

## C 级：低信任来源

不能作为事实依据。

- 自媒体爆料
- 社交平台小作文
- 未注明来源的截图
- 群聊截图
- 只说“据传”的内容
- 没有出处的表格或名单

使用方式：

- 只能标注为“待验证”。
- 如果只有 C 级来源，结论必须写“暂无可靠证据”。
- 不要把 C 级来源做成确定性视频文案。

## 判定规则

- S 级优先。
- A 级可辅助。
- B 级只能作为线索。
- C 级不能作为事实依据。
- 如果没有 S 级或 A 级来源，需要提示“证据质量不足”。
- 如果只有 C 级来源，必须提示“不能作为事实依据”。
- 如果多个来源互相矛盾，必须列出矛盾点，不能强行下结论。

- A级：

- C级：

## 证据质量判断
结论：

## 不确定点
1.

3.
